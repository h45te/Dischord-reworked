# Dischord-Reworked

## Description

Fork of Dischord (by Raclette3)  
The following changes have been made:
- Support for ES2024, discord.js (v14), and @discordjs/voice
- Changed the TOKEN reference to .env

## Requirement

- Node.js

- FFmpeg (must be available in PATH)

## Usage

### Run Bot

1. Copy `.env.example` as `.env` and put your bot token in `TOKEN`

1. Run `npm install`

1. Run `npm run build`

1. Run `npm start`

1. Yes! Party Time!!

### Bot Usage

Check out usage by sending `dc!help` to Dischord.
